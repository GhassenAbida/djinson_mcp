# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to **support@djinson.tn**. All security vulnerabilities will be promptly addressed.

Please do **not** disclose the vulnerability publicly until it has been addressed.
